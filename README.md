# ElixirTtt

An unbeatable command-line game of tic tac toe.

## Prerequisites

[Install elixir](http://elixir-lang.org/install.html) (on OS X with Homebrew: `brew install elixir`).

## How to Run

`mix escript.build`

`./elixir_ttt`

## How to Test

`mix test`

## Versions

Elixir 1.1.1

Erlang/OTP 18
